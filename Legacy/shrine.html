Luminon Shrine 소개

Shrine은 간단한 HTML 작성으로도 미려하고 가독성 높은 웹 포스트를 만들 수 있는 CSS입니다. 주로 글 작성에 사용되는 h1등을 비롯한 제목 태그와 p,a,li등 본문에 사용되는 태그, code등 기타 태그를 지원하며, 별도의 클래스 지정을 통해 버튼이나 링크등에 효과를 줄 수 있습니다.

사용 방법

Luminon_Shrine.css를 사용할 html에 연결해 사용합니다. snslink등의 속성은 별도의 css 수정이 필요하기 때문에 해당 속성을 수정할 css를 만들어 두시면 좋을 것 같습니다. 추가적인 사용 방법은 css내부에 주석으로 설명이 적혀 있습니다.
